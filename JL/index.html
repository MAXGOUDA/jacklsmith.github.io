Desktop: Notes

  • Generally more art direction. Images have been cropped and masked to suit themes and content. Exact layout varies from section to section.
  • Far more variety with the typography (potentially too much, but this is a visual design exploration!)
  • Staggered animation guides/encourages the reader down the page.
  • Shopping elements included via inline headers/links, and hover/tap tooltips on each image. This is a large project in itself, and can be explored far more.
